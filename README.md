# RITA project

Naming conventions:
Sentences are either: 
- ambiguous (AM) or non-ambiguous (NA)
* object first (OF) or subject first (SF)
* irregular rhythm (IR) or regular rhythm (RR)
example: AMOFIR
* ambiguous
* object first
* irregular rhythm
